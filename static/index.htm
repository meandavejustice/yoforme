yofor.me

An API for easily sending short messages through Yo.

POST http://yofor.me/{target}/{message}

Use it for:

  • Reminders
  • Notifying yourself when long-running tasks complete
  • Annoying your friends
  • Impressing your loved ones
  • Evil jk please don't :(

Example usage

cURL

Send a message to contra saying hey hi whats up:

$ curl -X POST http://yofor.me/contra/hey%20hi%20whats%20up

Notify via Yo when a build is done:

$ make && curl -X POST http://yofor.me/l33th4x/build%20is%20done\!

node

With request:

var request = require('request') request.post('http://yofor.me/contra/thanks%20for%20gulp', function(err) { if (err) { console.log('Error sending yo!') } else { console.log('Yo sent!') } })
